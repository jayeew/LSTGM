# LSTGM

This repository contains a PyTorch implementation of ["Long Short-Term Graph Memory Against Class-imbalanced Over-smoothing"](http://yangliang.github.io/pdf/mm23-lstgm.pdf).

## Runtime Environment

* CUDA=11.5
* torch=1.11.0
* torch-geometric=2.0.3

## Run 

```
python main.py
```
